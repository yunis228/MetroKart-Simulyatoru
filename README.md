# MetroKart Simulyatoru

Bu layihə Python dilində hazırlanmış bir MetroKart idarəetmə sistemidir.

## Əsas Funksiyalar:
* **Təhlükəsiz Giriş:** 3 cəhd haqqı ilə PIN kod yoxlanışı.
* **Balansın İdarə Edilməsi:** Borcların ödənilməsi və günlük limit (100 AZN) yoxlanışı ilə balans artırma.
* **Endirim Sistemləri:** Gediş sayına görə pilləli endirimlər və xüsusi rejimlər (Tələbə, Pensiyaçı).
* **Təcili Keçid:** Balans kifayət etmədikdə borcla keçid imkanı.
* **Statistika və Tarixcə:** Günlük gedişlərin hesabatı və son əməliyyatların siyahısı.

## Necə istifadə etməli:
1. Faylı yükləyin və Python mühitində işə salın.
2. PIN kodu daxil edin (Default: `1979`).
3. Menyudakı rəqəmləri seçərək əməliyyatları icra edin.
